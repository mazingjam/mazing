# Bok 01: Godnatt, lilla stjarna

Forsta utkast for GrowUps manad 1-bok. Boken ar tankt som en liten personlig godnattbok for nyfodda, med lag textmangd, mjuk rytm och bilder som kan anpassas efter barnets namn och avatar.

## Filer

- `manus.md` - sid-for-sid-manus med text, bildplacering och personaliseringsfalt.
- `production-brief.md` - format, ton, variabler, produktionsnoter och nasta steg.
- `image-prompts.md` - promptarna som anvandes for de genererade bildutkasten.
- `images/` - forsta genererade bildutkast.
