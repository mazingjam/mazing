# D&D Light

A lightweight visual fantasy adventure webapp prototype.

The first vertical slice focuses on mobile/iPad character creation:

- choose male/female presentation
- choose visual style
- choose race
- choose class
- name and add heroes to a four-character party
- start a placeholder adventure scene with the saved party
- choose Swedish or English before entering the app
- switch from the party scene to a portrait fullscreen clickable fantasyland map
- begin at a starting marker with two first travel options
- travel to named locations with their own environment backgrounds
- resolve three-choice encounters that grant gold, items, and stat gains
- preview the current hero against an adventure background
- use bottom controls for touch-first operation

The current character art set contains one transparent PNG for every race/class/gender/style combination:

- 4 races
- 4 classes
- 2 gender presentations
- 4 visual styles
- 128 generated portraits total

This is intended to grow into a simple cooperative adventure webapp with world exploration, NPCs, quests, combat, loot, and party loot splitting.
