# D&D Light

A lightweight visual fantasy adventure webapp prototype.

The first vertical slice focuses on mobile/iPad character creation:

- choose race
- choose gender presentation
- choose class
- choose color identity
- preview the current hero against an adventure background
- use bottom controls for touch-first operation

This is intended to grow into a simple cooperative adventure webapp with world exploration, NPCs, quests, combat, loot, and party loot splitting.
