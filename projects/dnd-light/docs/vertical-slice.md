# Vertical Slice

## Goal

Create the first playable-feeling surface for D&D Light: character creation.

## Current Features

- Mobile-first character creation screen.
- Visual hero preview.
- Gender, style, race, and class choices.
- Bottom navigation buttons.
- Randomize button.
- Simple derived stats.
- Transparent cartoon-style character portrait for every race/class/male-female/style combination.

## Next Systems

- Party roster.
- First world node.
- NPC encounter.
- First quest.
- Simple combat.
- Loot reward and party split.
