# Dictation 2026-06-15 006: Bribe

## Raw Notes

- Another mechanic: `Bribe X`.
- Bribe is a cost defined by X.
- Example bribe costs can be mana costs, gold costs, or sacrificing a permanent.
- Any opponent may pay the bribe cost.
- If an opponent pays the bribe cost, it can "fizzle" any action from a card with Bribe.
- On a creature, Bribe can be paid when it tries to attack, block, or activate/trigger an ability.
- On an instant, Bribe can apply to the spell effect. Example: "draw 2 cards" is ignored if the bribe is paid.
- A card can have a replacement-style bribe outcome instead. Example: "Deal 3 damage to any target. If the bribe cost was paid, deal 1 damage instead."
- Bribe may still work as shorthand that prepends a defined cost to a specific bribe outcome.
- Example: `Bribe {Gold}: When this attacks, tap it instead.`

## Immediate Interpretation

- Bribe is an opponent-facing interaction mechanic.
- Bribe creates a payment window for opponents before specific actions or effects happen.
- Bribe does not necessarily counter the spell or ability; it can replace, reduce, or cancel the relevant action/effect.
- Bribe costs are flexible and card-defined.
- Bribe needs careful templating because "any action" is broad.
- Bribe can be keyword-like if the keyword defines the opponent payment permission, while the card text defines the consequence.

## Possible Shorthand Model

> Bribe [cost]: [bribe outcome].

Meaning:

> Any opponent may pay [cost] at the relevant time. If a player does, [bribe outcome].

Example:

> Bribe `{Gold}`: When this attacks, tap it instead.

## Rules Risks

- "Any action" is too broad as final rules text.
- Need clear event windows: attacking, blocking, triggered ability resolution, activated ability activation/resolution, spell resolution.
- Need to define which opponent gets priority to pay in multiplayer.
- Need to define whether multiple opponents can pay, and what happens if more than one tries.
- Need to define whether paying a bribe targets anything. Current assumption: no.
- Need exact grammar for Bribe shorthand, especially for attacks, blocks, triggered abilities, and spell effects.
