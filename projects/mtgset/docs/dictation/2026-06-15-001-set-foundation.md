# Dictation 2026-06-15 001: Set Foundation

Raw/near-raw notes from spoken brainstorming.

## Raw Notes

- It is a three-color set with guilds/factions like Ravnica, but all of the guilds exclude green.
- Green has its own dual-color theme for all colors, and also a mono-color path.
- The set is draft-first.
- There are two new card types.
- One of the new card types is `Building`.
- There will be many new keywords, to be defined later.
- Start with this foundation and refine from here.

## Immediate Interpretation

- The main faction structure is not the usual two-color Ravnica guild model. It sounds like the primary guild/faction identities are three-color and intentionally non-green.
- Green is structurally special: it may be outside the main faction system, acting as a bridge/splash/environment color, with both dual-color pairings and a mono-green route.
- Draft architecture should be designed early, because faction/color structure is unusual and can easily create color imbalance.
- `Building` needs rules definition as a new card type.
