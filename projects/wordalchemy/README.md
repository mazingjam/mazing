# Word Alchemy MVP

A small mobile-friendly web MVP for a puzzle/word game.

## Concept

Build compound words from meaning fragments. Some fragments also change the board:

- `FIRE` clears one cracked tile.
- `WATER` restores a used fragment.
- `GHOST` lets one locked tile be used.
- `ROOT` refreshes one nature fragment.

## Run

Open `index.html` directly, or serve the folder:

```powershell
powershell -ExecutionPolicy Bypass -File .\start-server.ps1
```
