# Sunder old cards

Archived reference material for the earlier Sunder card prototype.

These images are kept as historical design input, not as the current source of truth.
Use them to recover mechanics, naming, visual direction, icon language, and balance
ideas when building the new board game structure.

## Split Card Catalog

The sheet images have also been split into individual card references:

- `cards/images/` - one cropped image per visible card
- `cards/cards.csv` - structured catalog data
- `cards/index.md` - categorized card index
