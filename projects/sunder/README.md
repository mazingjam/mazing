# Sunder

Sunder is a cooperative fantasy board game project inspired by deckbuilding dungeon crawlers such as Slay the Spire: The Board Game, but rebuilt around a different resource model and a physical book-map board.

The current design direction:

- Players control heroes with asymmetric energy profiles.
- Energy comes in red, green, blue, white, and black.
- The map is a large physical book. Each open spread is a board state with encounters, opportunities, branches, and sometimes split-party paths.
- Cards, heroes, encounters, and book pages should be designed as a connected campaign-like run system rather than as isolated components.

## Project Structure

- `docs/` - high-level design decisions, glossary, and open questions.
- `design/` - mechanical design documents.
- `cards/` - card design notes, templates, and future card data.
- `board-book/` - book-map structure, spread ideas, and encounter page planning.
- `old-cards/` - archived images from the older card prototype.
- `playtests/` - playtest logs and version notes.
- `assets/` - future art, icons, diagrams, and printable files.
- `prototypes/` - future digital or print-and-play prototypes.
