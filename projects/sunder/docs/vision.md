# Vision

Sunder is a new take on the cooperative roguelike deckbuilding board game.

The target is not to clone Slay the Spire. The useful inheritance is the run structure: heroes build decks, make route decisions, fight encounters, gain rewards, and adapt to increasingly dangerous situations. Sunder should add a more tactile board presence, richer party coordination, and a resource system that creates identity and tension without becoming bookkeeping-heavy.

## Core Pillars

1. Color-driven energy identity
   Heroes and decks are shaped by red, green, blue, white, and black energy rather than a single generic energy source.

2. The board as a book
   The map is a physical book with thick pages. Opening a spread creates the current board: encounters, opportunities, terrain, branching choices, and sometimes multiple simultaneous paths.

3. Cooperative tactical pressure
   Players should coordinate without requiring perfect shared turns. Splitting the party should sometimes be risky, sometimes correct, and often tempting.

4. Compact but expressive cards
   Cards should be easy to parse at table speed, but the system should support strong synergies, color identity, and memorable hero builds.

5. Run-based progression
   A session should feel like a run through a living location, not a static scenario. The book should make route choices concrete.

## Current Assumptions

- Players have hero cards that provide base energy each turn.
- Cards may add temporary or persistent access to more energy.
- White energy can be spent as any color.
- Black energy can only be paid with black energy.
- The color system should create deckbuilding constraints, not rigid class locks.
