# MVP Scope

The MVP should be small, playable, and emotionally clear. It should prove the Eternal Soul fantasy before expanding content.

## Target Experience

A player should be able to complete a 10-15 minute run, lose or retire a hero, and see that hero become part of future runs.

The first milestone is successful if the player can say:

> I remember what happened to that hero, and I want to see them again.

## Included

### Heroes

Start with 3 heroes:

1. Warrior
   - Role: durable frontline scaling through damage taken.
   - Build identity: armor, retaliation, rage, ally protection.
2. Necromancer
   - Role: summons and death-trigger scaling.
   - Build identity: skeletons, sacrifice, curses, soul economy.
3. Alchemist
   - Role: volatile utility and status effects.
   - Build identity: poison, bombs, healing, random mixtures.

### Skills

20 base skills total.

Each hero should have:

- 3 hero-specific skills
- Access to a shared neutral pool
- 1 signature starting skill

### Modifiers

15 skill modifiers.

Examples:

- Chain
- Split
- Burn
- Pierce
- Echo
- Giant
- Lifesteal
- Fragile
- Poisoned
- Quickened

The MVP should support combining at least one base skill with two modifiers.

### Encounters

30 encounter definitions:

- 16 combat
- 4 elite
- 3 boss
- 3 merchant
- 2 treasure
- 2 Eternal Soul encounter

### Eternal Souls

When a hero dies or completes a run, create an Eternal Soul record.

The soul stores:

- Name
- Hero class
- Level
- Cause of death or ascension
- Highest damage dealt
- Kills
- Skills used
- Modifiers used
- Defining trait
- Legacy trait
- Relationship hooks

### Legacy Uses

MVP Eternal Souls can return in 3 ways:

1. Recruitable companion
2. Passive mentor blessing
3. Event character

Boss/rival/legend status can wait until after the MVP unless it is cheap to implement.

## Not Included

Avoid in the first prototype:

- 25 heroes
- Complex equipment rarity system
- Full passive skill forests
- Long-term economy tuning
- Online features
- Procedural narrative prose beyond short event text
- High-cost art production

## Success Criteria

The MVP is working when:

- A run can start with a choice of 3 heroes.
- Combat resolves automatically in 5-20 seconds.
- The player makes a decision every 10-20 seconds outside combat.
- A hero can die and become an Eternal Soul.
- An Eternal Soul can affect a later run.
- At least one build feels meaningfully different because of skill modifiers.
