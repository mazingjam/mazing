# Eternal Heroes

Eternal Heroes is a mobile-first roguelite strategy game about building a dynasty of legendary heroes.

The core promise:

> You are not only building a character. You are building a pantheon.

Each run creates permanent history. Heroes who die are preserved as Eternal Souls and can later return as companions, mentors, rivals, event figures, bosses, or legends.

## Design Pillars

1. Meaningful decisions often
2. Short autobattler combat
3. Deep skill and modifier builds
4. Permanent hero legacy without flat power creep

## MVP Goal

The first prototype should prove one thing:

> A dead hero can become more interesting after death than before it.

The MVP should include 3 heroes, short combat encounters, reward choices, and a first version of the Eternal Soul system.
